# CSS Shape Animations

A collection of animated geometric shapes built using pure HTML and CSS.
No JavaScript. No external libraries or frameworks required.

---

## Live Demo

[View Demo](https://mithanya.github.io/css-shape-animations/shapes.html)

## Shapes Included

| Shape | Animation Type |
|-------|---------------|
| Circle | Continuous rotation |
| Star | Continuous rotation |
| Triangle | Ease-in-out rotation |
| Hexagon | Rotation with pulse |
| Blob | Rotation with morph |
| Diamond | Fast continuous rotation |
| Pentagon | Rotation with pulse |
| Ring Spinner | Multi-color spin |
| Square | Rotation with scale |

---

## CSS Concepts Used

| Concept | Description |
|---------|-------------|
| @keyframes | Defines animation steps |
| transform: rotate() | Rotates shapes continuously |
| transform: scale() | Scales shapes up and down |
| clip-path | Creates geometric shapes |
| border-radius | Morphs blob shape |
| box-shadow | Creates glow effect on shapes |
| text-shadow | Creates glow effect on labels |
| animation-delay | Staggers floating effect |
| filter: drop-shadow | Glow on clipped shapes |

## Getting Started

**Clone the repository**
```bash
git clone https://github.com/Mithanya/css-shape-animations.git
```

**Open in browser**
```bash
cd css-shape-animations
open shapes.html
```

No installation required. Open directly in any modern browser.
